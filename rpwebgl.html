LampAndRock

WebGL Unity Demo - Ronak Parikh

Warning: This demo takes a long time to load, ignore all warnings and press wait if the page is unresponsive. If it doesn't work, try it in Microsoft Edge.

Controls: WASD to Move, X to Reset Position, Z to Adjust Lamp, Hold SPACE to Fly. You are the rock, you have gravity, the lamp does not.

I made this game in Unity and designed the models in C4D. The purpose of this game is to test how well my models from cinema4d can be rendered in unity. As you can see, all texture, normal, displacement, bump, and reflection maps are able to bake properly and interace with other objects and lighting conditions properly. The UV unwrapped maps work especially well and you can even see the dirt/dust on the rock! The C# code for unity is also escpecially easy to work with and is just like Java and Processing.
